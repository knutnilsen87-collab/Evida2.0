# User-Facing Copy Glossary v1

Default UI language is Norwegian.

## Tone

Use:
- calm
- precise
- useful
- direct
- non-technical
- legally careful

Avoid:
- technical jargon
- dramatic warnings
- vague AI hype
- generic support phrases
- too much text

## Button copy

Use:
- «Vis meg»
- «Forklar enklere»
- «Gå til første problem»
- «Fiks sammen med meg»
- «Sjekk arbeidet mitt»
- «Forhåndsvis eksport»
- «Send til godkjenning»
- «Opprett eksportpakke»
- «Legg til kilde»
- «Se dokumentet»

Avoid:
- «Execute»
- «Run agent»
- «Validate source refs»
- «Trigger export gate»
- «Submit job»

## Common messages

### Missing source
> «Denne påstanden mangler kilde.»

### Export blocked
> «Eksport er stoppet fordi noe må kontrolleres først.»

### OCR uncertainty
> «Denne siden er usikkert lest. Se over teksten før du bruker den som kilde.»

### Document changed
> «Dokumentet ser ut til å være endret siden analysen ble gjort.»

### Approval required
> «Dette må godkjennes før det kan utføres.»

### No access
> «Du har ikke tilgang til denne saken.»

### Assistant legal boundary
> «Jeg kan hjelpe deg å kontrollere kilder, struktur og risiko, men endelig juridisk vurdering må gjøres av ansvarlig jurist.»

## Assistant microcopy

### Empty assistant
> «Spør meg om hva du ser på, hva som mangler, eller hva du bør gjøre videre.»

### Blocked state
> «Jeg kan vise deg nøyaktig hva som stopper dette.»

### Confused user
> «La oss ta det enklere. Jeg viser deg første steg.»

### Repeat question
> «Dette ligner på det vi nettopp så på. Forskjellen her er ...»

### Guided flow
> «Jeg kan gå gjennom dette med deg steg for steg.»
